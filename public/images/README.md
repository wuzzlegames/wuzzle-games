# Header and UI icons

Expected image filenames for the site header (from `public/images/`). If a file is missing, the header shows a fallback icon.

- `logo.png` – Site logo (click = home)
- `leaderboard.png` – Leaderboard
- `notifications.png` – Notifications
- `hamburger.png` – Menu
- `sign-in.png` – Sign in
- `sign-out.png` – Sign out
- `friends.png` – Friends
- `challenges.png` – Challenges

All header icons use a fixed size (e.g. 32×32) for alignment.

## Home page mode row icons

Used on the Single Player grid. If missing, a fallback letter is shown.

- `Daily Standard.png` – Daily (standard)
- `Daily Speedrun.png` – Daily (speedrun)
- `Solution Hunt Standard.png` – Solution Hunt (standard)
- `Solution Hunt Speedrun.png` – Solution Hunt (speedrun)
- `Marathon Standard.png` – Marathon (standard)
- `Marathon Speedrun.png` – Marathon (speedrun)

## Home page multiplayer icons

Used on the Multiplayer grid. If missing, a fallback letter is shown.

- `Join Room.png` – Join Room
- `Host Room.png` – Host a Room
- Challenge a Friend and Ranked use fallback letters (C, R) only.
